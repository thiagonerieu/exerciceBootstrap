FONTLOG for Banana
-------------------

This file provides detailed information on Banana Font.
This information should be distributed along with the font and any derivative works.


Basic Font Information
--------------------------

It is advised to use the SingleLine file to make any modification, then, if needed, to save a second "flattened" file where you "expand strokes" in order for it to properly work in printed and web designs.
You can create as much different penstrokes as you want for a variety of results. You can also contact Clara Bougon if you need access to the original Metapost code to modify it.
